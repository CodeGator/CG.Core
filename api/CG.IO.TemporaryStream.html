    public class TemporaryStream : Stream, IAsyncDisposable, IDisposable
    Remarks

    The idea, with this class, is to create a temporary stream mapped to an equally temporary backing store. In this case, a temp file. So, this is good to use when you only need storage for a short period of time. It's not good if you want the information to stick around since temp files really aren't guaranteed to live for any set period of time.
